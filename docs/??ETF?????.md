# 国内 ETF 监测池说明

更新日期：2026-07-19

## 定位

这 5 只产品构成的是“平衡型研究观察池”，不是要求立即买入的推荐清单。系统当前把持仓数量和成本设为 0，不会连接券商，也不会自动下单。产品只有在数据可用、评分连续两个不同交易日达到阈值、回测完成后，才会出现模拟提醒。

Alpha Vantage 免费层使用日线收盘数据，compact 通常只返回最近约 100 个交易日。系统因此在交易日北京时间 16:30 每日完成一次完整分析，约使用 5 次请求，并在连续请求之间等待 13 秒。交易时段另用 AKShare ETF 15分钟行情逐只做实验性风险观察，不消耗 Alpha Vantage 额度。

## 为什么是这 5 只

- 沪深300、中证500分别覆盖大盘与中盘风格，减少只押单一市值层级的偏差。
- 红利低波与创业板分别代表防御和成长两端，可以观察市场风格切换。
- 黄金不是股票指数，加入它是为了观察非权益资产的分散作用，而不是保证避险有效。
- 5 只产品每天约 5 次 API 请求，明显低于 Alpha Vantage 官方当前列示的免费层每日 25 次上限。

## 使用规则

1. 第一天达到 70 分只累计确认，不提醒；第二个不同交易日仍达到 70 分才生成模拟提醒。
2. 同一市场数据日期重复运行时，不累计确认次数，也不重复发信号。
3. 止盈、止损当前留空。只有录入你自己的真实成本、数量和纪律价格后，相关提示才有个人意义。
4. 每日评分可用于复盘与形成候选操作，但不能机械地每天调仓。实际调整前还要检查费用、税费、流动性、组合集中度与自身风险承受能力。
5. 免费 compact 日线样本较短，页面回测只能作为基本一致性检查，不能证明策略具有长期有效性。
6. AKShare 盘中快照聚合公开网页行情，可能延迟、中断或发生字段变化；仅作辅助观察，不能替代券商条件单。
